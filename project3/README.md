#  SQL E-Commerce Sales Analysis

This project was completed as part of my **Data Analytics Internship** using **MySQL Workbench**. The goal was to analyze an e-commerce orders dataset by writing SQL queries to retrieve, filter, sort, and summarize data for business insights.

##  Tools Used

* MySQL Workbench
* SQL

##  SQL Concepts Covered

* `SELECT`
* `WHERE`
* `ORDER BY`
* `COUNT()`
* `SUM()`
* `AVG()`
* `GROUP BY`

##  Analysis Performed

* Retrieved the complete orders dataset.
* Filtered delivered orders and credit card transactions.
* Identified high-value orders.
* Sorted orders by revenue and order date.
* Calculated total orders, total revenue, and average order value.
* Analyzed quantity sold and revenue by product.
* Counted orders by payment method and order status.
* Measured revenue generated through different referral sources.

##  Dataset

The dataset contains **1,200 e-commerce orders** with details such as:

* Order ID
* Order Date
* Product
* Quantity
* Payment Method
* Order Status
* Referral Source
* Total Price

##  Outcome

This project demonstrates practical SQL skills for querying, aggregating, and analyzing business data using MySQL Workbench to generate meaningful insights from an e-commerce dataset.

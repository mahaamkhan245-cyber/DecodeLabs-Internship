#  Data Cleaning & Preparation

## Project Overview

This project focuses on cleaning and preparing a raw e-commerce dataset for analysis. The primary data cleaning process was performed using Microsoft Excel, while an equivalent Python (Pandas) script was created for automation and GitHub portfolio purposes.

## Tools Used

* Microsoft Excel
* Python (Pandas)
* GitHub

## Data Cleaning Tasks Performed

* Standardized date format
* Auto-fitted row and column widths
* Aligned table headers
* Recalculated Total Price using Quantity × Unit Price
* Rounded Total Price to 0 decimal places
* Filled empty cells with "Null"
* Sorted records from oldest to newest date
* Highlighted repeat customers
* Highlighted null values
* Highlighted pending orders

## Files Included

* `Data Analytics (task1).xlsx` – Cleaned Excel dataset
* `data_cleaning.py` – Python (Pandas) script demonstrating the cleaning process

The dataset was cleaned primarily in Microsoft Excel. A Python (Pandas) script has been included to demonstrate how the same cleaning workflow can be automated programmatically.

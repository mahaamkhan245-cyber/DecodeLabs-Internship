#  E-Commerce Sales Dashboard (Power BI)

## Project Overview

This interactive **Power BI dashboard** was developed during my **Data Analytics Internship** to analyze and visualize e-commerce sales performance. Built using a dataset of **1,200 customer orders**, the dashboard enables stakeholders to monitor key business metrics, identify sales trends, and evaluate product and marketing performance through interactive visualizations.

---

##  Dashboard Features

### KPI Cards

*  **Total Revenue:** **$1.26M**
*  **Total Orders:** **1,200**
*  **Total Quantity Sold:** **~4,000 Units**

### Interactive Visualizations

* Revenue by Referral Source
* Orders by Payment Method
* Orders by Order Status
* Revenue by Product Category
* Monthly Revenue Trend
* Quarter Slicer for interactive filtering

---

## Dataset

The dashboard is built using an e-commerce sales dataset containing **1,200 customer orders** from **January 2023 to June 2025**.

**Dataset includes:**

* Order ID & Order Date
* Customer ID
* Product Category
* Quantity & Unit Price
* Total Price
* Payment Method
* Referral Source
* Order Status

---

##  Key Insights

* **Instagram** and **Email** generated the highest referral revenue.
* **Laptops** and **Chairs** were the top revenue-generating products.
* Payment methods were evenly distributed, indicating balanced customer preferences.
* Revenue peaked during **Month 5**, highlighting a strong mid-year sales period.
* Order statuses were evenly distributed, suggesting opportunities to reduce cancellations and returns.

##  How to Use

1. Open the `.pbix` file in **Power BI Desktop**.
2. Use the **Quarter slicer** to filter dashboard data.
3. Hover over visuals to view detailed tooltips.
4. Select chart elements to cross-filter related visualizations.
